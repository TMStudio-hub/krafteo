# Krafteo V5.2

Versión lista para GitHub Pages.

## Subir a GitHub
1. Sube `index.html` al repositorio `krafteo`.
2. Haz Commit changes.
3. GitHub Pages actualizará la app.

## Cambios
- UI más limpia y navegación simplificada.
- Ajustes unificados: Insights, Sync, Backup y App.
- Crear cliente nuevo desde pedido.
- Recibo con copia cliente y copia taller.
- Estado automático.
- Animación DRACARYS al completar pedido.
- Logo de Taller de Dragones integrado.
